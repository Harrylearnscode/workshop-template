### Week 5 Achievements:

* **Regarding Database & Caching:**
  * **DynamoDB:** Understood NoSQL architecture, concepts of Tables, Items, Attributes, and pricing models (Provisioned vs On-demand).
  * **ElastiCache:** Grasped the role of Caching in improving application performance and common engines (Redis/Memcached).

* **Regarding Monitoring & Governance:**
  * **CloudWatch:** Learned how to set up Metrics, create monitoring Dashboards, and configure Alarms for incident alerting.
  * **CloudTrail:** Understood how this service records API call history for auditing and security purposes.
  * Completed practical Labs on setting up system monitoring.

* **Regarding Container Technology:**
  * **Docker:** Mastered basic concepts (Image, Container, Dockerfile) and the application packaging process.
