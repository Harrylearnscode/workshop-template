**In this page**, how will you introduce your worklog? How many weeks did it take to complete the program? What did you do during those weeks?

Typically and as a standard, a worklog is carried out over approximately 3 months (during the internship period) with the weekly content as follows:

**Week 1:** [Getting started with AWS, IAM, EC2, VPC and environment setup](1.1-week1/)

**Week 2:** [Advanced Networking (VPN), Storage (S3), Database (RDS) and project initialization](1.2-week2/)

**Week 3:** [Frontend Development (Menu, Cart), Backend Integration and Monitoring with CloudWatch](1.3-week3/)

**Week 4:** [Finalizing Authentication (Auth), learning Lightsail, Auto Scaling and AWS CLI](1.4-week4/)

**Week 5:** [NoSQL Database (DynamoDB), Caching (ElastiCache), Docker and Monitoring](1.5-week5/)

**Week 6:** [Finalizing Order function, Advanced Docker and AWS Skill Builder learning](1.6-week6/)

**Week 7:** [Completing AWS theory, reviewing](1.7-week7/)

**Week 8:** [Practicing real exam questions and assessing capability via mock tests](1.8-week8/)

**Week 9:** [Finishing AWS Skill Builder path, developing Admin Meal Management page](1.9-week9/)

**Week 10:** [Finalizing Admin Order Management, researching and drafting Architecture diagram](1.10-week10/)

**Week 11:** [Optimizing Backend (Payment), improving UX/UI and attending Workshop](1.11-week11/)

**Week 12:** [Learning GitLab, finalizing Architecture and Deploying the project](1.12-week12/)
